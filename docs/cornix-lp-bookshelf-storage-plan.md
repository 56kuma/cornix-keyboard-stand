# Cornix LP 短辺下向き Bento Dock 設計計画

作成日: 2026-06-16

対象: JezailFunder Cornix LP 無線分割キーボードの左右ユニットを、本棚の本のように並べ、短辺を下にして縦置き収納するための 3D プリントスタンド。

## 1. 今回作るもの

左右の Cornix LP を 1 枚ずつ縦に差し込める、2 スロットの卓上収納スタンドを作る。

今回の要件:

- 本棚の本のように左右ユニットを並べる。
- キーボードの短辺を下にして収納する。
- 遊びをなくし、できるだけぴったり入る寸法にする。
- 見た目の洗練さ、かっこよさを維持する。
- キーボード本体を加工しない。
- CornixHub のアクセサリ設計を参考にする。

公開ページに Cornix LP 本体の外形寸法図は見つからないため、現在の STL は仮寸法で生成している。実機で本当にぴったり合わせるには、短辺長さと厚みを実測して生成スクリプトに入れる。

## 2. 参照情報

公式ページで確認した前提:

- Cornix LP は 3x6 配列の分割キーボード。
- 筐体は 6063 アルミ削り出し。
- 6 度、12 度、18 度、24 度の角度調整機構がある。
- USB-C 有線接続と Bluetooth に対応する。

追加で参照した CornixHub アクセサリ情報:

- CornixHub の非公式アクセサリ一覧には、Cornix LP 用のパームレスト、カバー兼リストレスト、ハードキャリーケースなどが掲載されている。
- Cornix Bento Box は、Cornix LP 専用ハードキャリーケースとして `W188 x D120 x T55 mm` と記載されている。
- Cornix Bento Box は PETG-GF を使い、ケース本体とカバーをマグネットで固定する設計。
- Cornix Bento Box はキーボード同士を向かい合わせで収納し、デフォルトキーキャップとエンコーダー間のクリアランスは約 1.0 mm と説明されている。
- Cornix LP 用カバー兼リストレストは、ズレにくくするため片側にフック状の部分を追加し、クリアランスをかなり詰めた設計思想になっている。

## 3. 再設計方針

CornixHub のアクセサリ情報から、次の考え方を反映する。

| 参照した要素 | 今回の設計への反映 |
| --- | --- |
| Cornix Bento Box の `D120 x T55 mm` | スタンド外形を奥行き 120 mm、高さ 55 mm に寄せる |
| 向かい合わせ収納と約 1.0 mm クリアランス | 低遊びの寸法駆動スロットにする |
| 個体差ではめ合いが変わる注意書き | `KEYBOARD_SHORT_EDGE` と `KEYBOARD_THICKNESS` を実測差し替え前提にする |
| カバー兼リストレストのフック状ズレ止め | 前後ストッパーに 1.0 mm の低いフック形状を入れる |
| 3D プリント一体成形アクセサリ | サポートなしで印刷しやすい一体形状を維持する |

## 4. 現在の寸法

生成済み STL の仕様:

| 項目 | 値 |
| --- | ---: |
| ベース幅 | 74.164 mm |
| ベース奥行き | 120.0 mm |
| ベース厚み | 6.0 mm |
| 全体高さ | 55.0 mm |
| スロット数 | 2 |
| 想定キーボード短辺 | 96.0 mm |
| 想定キーボード厚み | 24.0 mm |
| 総クリアランス | 0.5 mm |
| スロット内寸 長さ | 96.5 mm |
| スロット内寸 幅 | 24.5 mm |
| 側面ブレード高さ | 49.0 mm |
| 前後ストッパー高さ | 13.0 mm |
| フックかかり量 | 1.0 mm |
| STL 三角形数 | 172 |

ベースは `120.0 / 74.164 = 1.618` で、長辺と短辺が黄金比になるようにしている。

## 5. ぴったり合わせるための実測項目

実機をノギスで測り、次の 2 箇所をスクリプトに入れる。

| パラメータ | 測るもの | 現在値 |
| --- | --- | ---: |
| `KEYBOARD_SHORT_EDGE` | 収納時に下へ向ける短辺の外形長さ | 96.0 mm |
| `KEYBOARD_THICKNESS` | キーキャップを含む本体厚み、または収納時に溝へ入る厚み | 24.0 mm |

きつすぎる場合だけ `FIT_CLEARANCE` を増やす。FDM 3D プリントでは、総クリアランス `0.5 mm` はかなり詰めた値。プリンターの精度が高い場合は `0.4 mm`、初回試作なら `0.8 mm` も候補になる。

## 6. 形状と意匠

美観と保持力のため、次の形状を入れている。

- 短辺を下にする 2 連ポケット。
- 本体厚み方向を受ける左右ブレード壁。
- 中央の山型ディバイダー。
- 短辺方向の遊びを止める前後フックストッパー。
- 角を落とした黄金比台座。
- スロット床面の薄い接触レール。
- 黄金比位置の外側補強リブ。
- Cornix Bento Box の外形感に寄せた `D120 x T55 mm` 級のコンパクトなプロポーション。

黒、濃いグレー、マットシルバー系で印刷すると Cornix LP のアルミ筐体と合わせやすい。実機に傷を付けたくない場合は、接触面へ薄いフェルトや TPU シートを貼る。ただし貼る厚みの分だけ `KEYBOARD_THICKNESS` または `FIT_CLEARANCE` を調整する。

## 8. 印刷設定案

推奨:

- 材料: PETG、PETG-CF、PETG-GF、または PLA+
- レイヤー高: 0.20 mm
- ノズル: 0.4 mm
- 壁数: 4
- インフィル: 25% から 35%
- 上下面: 5 層以上
- ブリム: 反りが出る場合のみ使用

ぴったり入れる前提なので、初回は積層誤差を見るために 10 mm 高さ程度で途中停止または部分試作して、スロット幅を確認すると安全。

## 9. 実機確認ポイント

印刷前に確認すること:

- 短辺がスロット底面に入る向きになっているか。
- スロット内寸 `96.5 x 24.5 mm` が実機に対して適切か。
- USB-C ポートや電源スイッチがブレード壁に当たらないか。
- キーキャップを強く押し付ける向きになっていないか。
- 2 枚を入れたとき、中央ディバイダーに干渉しないか。
- 前後フックストッパーで取り出しが重くなりすぎないか。

実機で合わない場合の調整:

- 短辺方向がきつい: `KEYBOARD_SHORT_EDGE` を実測値にするか、`FIT_CLEARANCE` を増やす。
- 厚み方向がきつい: `KEYBOARD_THICKNESS` を実測値にするか、`FIT_CLEARANCE` を増やす。
- 倒れやすい: `SIDE_WALL_HEIGHT` を増やす。
- 取り出しにくい: `SIDE_WALL_HEIGHT` を下げるか、`END_STOP_HOOK_OVERHANG` を減らす。
- 机上の存在感を強めたい: `BASE_DEPTH` を 120 mm のまま、`BASE_CHAMFER` を少し増やす。
